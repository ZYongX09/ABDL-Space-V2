# 更新日志

## [v2.7.0] - 2026-05-19

### ✨ 新增

- **发帖图片上传**：发帖时支持上传图片，含预览网格与全屏查看
- **私信功能**：新增私信系统（开发调试中）

### 🔧 优化

- **移动端布局**：全面优化移动端页面布局与交互体验
- **个人中心**：优化个人中心页面展示
- **性能与稳定性**：提升产品性能，修复已知问题

## [v2.6.1] - 2026-05-18

### ✨ 新增

- **关于页**：新增 GitHub 项目和开发者博客入口按钮
- **Cloudflare Pages**：支持部署到 Cloudflare Pages

## [v2.6.0] - 2026-05-17

### ✨ 新增

- **图片上传组件**：新增 ImageUploader 和 ImageGrid 图片上传与网格预览组件

### 🔧 优化

- **CAPTCHA 频率感知触发**：验证不再每次都弹，改为智能触发
  - 发帖/评论/删除等操作：2 分钟内首次操作免验证，连续操作才弹 CAPTCHA
  - 登录页：前 2 次失败不弹验证，第 3 次起才出现 CAPTCHA（防暴力破解）
  - 注册页：保持始终验证（防批量注册）
  - 验证通过后重新计时，不影响后续正常使用

## [v2.5.1] - 2026-05-16

### ✨ 新增

- **返回按钮**：新增圆形半透明返回上一页按钮，移动端固定在页面左上角，PC 端位于内容区左上；智能判断来源页，从固定 tab 跳转或外部进入时自动隐藏
- **首页排行入口**：移动端纸尿裤列表页新增「排行」按钮，点击跳转排行榜页面（PC 端不显示）
- **百度统计**：接入百度统计，全站生效
- **捐赠条款**：用户协议新增第五条「捐赠」，明确自愿赠与、不可退还、不赋予特殊权利
- **数据收集条款**：用户协议新增第六条「数据收集与第三方服务」，披露百度统计数据收集

### 🔧 优化

- **移动端底部菜单栏**：移除选中 tab 下方的蓝色色块，改为图标和文字直接变蓝
- **排行榜布局**：优化最强吸收列表的移动端布局，长吸收量数据不再遮挡纸尿裤名称
- **账户切换面板**：侧边栏中账户切换移至设置按钮上方；下拉面板改为高模糊毛玻璃效果，适配深色/多彩主题

### 📄 文档

- **隐私政策**：更新日期至 2026-05-16；移除「无分析数据」声明，新增百度统计说明（2.4 节）；第三方服务列表新增百度统计
- **用户协议**：更新日期至 2026-05-16；条款重新编号（原第五至第十条顺延为第七至第十二条）
